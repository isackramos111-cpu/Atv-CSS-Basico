# 🎮 Atividade — Ficha Técnica com CSS

## O que você vai fazer

Você recebeu uma página HTML pronta: uma **ficha técnica de personagem**.
Sua missão é criar o arquivo `estilo.css` para deixar essa página com a sua cara!

Você pode escolher **qualquer personagem** — de games, animes, filmes, séries, livros ou até um personagem criado por você.

---

## Arquivos da atividade

| Arquivo | O que é |
|---|---|
| `ficha-personagem.html` | Estrutura da página — **não altere este arquivo** |
| `estilo.css` | Arquivo CSS que você vai preencher |

> ⚠️ Não edite o HTML. Todo o seu trabalho deve estar no `estilo.css`.

---

## Passo a passo

### 1. Escolha seu personagem
Pense em um personagem que você gosta. Pode ser de qualquer universo!

### 2. Preencha o HTML com os dados do personagem
Espera — mas você disse que não devo editar o HTML!

Sim, mas há uma exceção: **os textos e imagens de conteúdo**. Você pode (e deve) substituir os textos de exemplo pelos dados reais do seu personagem, como nome, descrição, valores de atributos e habilidades.

O que **não pode mudar** são as **tags, IDs e classes** — esses são os "ganchos" que o seu CSS vai usar.

Mas se for mudar ou adicionar mais tags HTML, saiba o que está mudando e adicione o CSS equivalente.

### 3. Encontre imagens
Procure imagens do seu personagem na internet (arte oficial, fanart com crédito, ou crie a sua).
Substitua os endereços das imagens no HTML pelas imagens que você encontrou.

### 4. Crie o CSS
Abra o arquivo `estilo.css` e comece a estilizar!
Siga os requisitos abaixo.

---

## Requisitos obrigatórios

Você **precisa** usar todas as propriedades abaixo no seu CSS. Marque cada uma conforme for completando:

### Cores
- [ ] `color` — cor do texto de todos os elementos
- [ ] `background-color` — cor de fundo de todos os elementos

### Fontes
- [ ] `font-family` — escolha uma fonte para a página (pode usar Google Fonts!)
- [ ] `font-size` — tamanhos diferentes para título, subtítulo e texto normal
- [ ] `font-weight` — use negrito (`bold`) em pelo menos um elemento

### Bordas
- [ ] `border` — adicione borda de todos os elementos
- [ ] `border-radius` — deixe algum elemento com cantos arredondados

### Espaçamento
- [ ] `margin` — espaço externo de todos os elementos
- [ ] `padding` — espaço interno de todos os elementos

### Tamanho
- [ ] `width` — defina a largura de todos os elementos

## Desafios extras (bônus, não obrigatório)

Quer ir além? Tente implementar:

- 🌟 Usar uma fonte do **Google Fonts** (`fonts.google.com`)
- 🌟 Deixar a página com um visual temático do personagem (cores, estilo)
- 🌟 Usar `text-align` para centralizar elementos
- 🌟 Usar `text-transform` para deixar o nome do personagem em maiúsculas
- 🌟 Usar `opacity` para criar um efeito sutil em algum elemento
- 🌟 Adicionar `:hover` em algum card para ele reagir ao mouse

---

## Como usar o Google Fonts

1. Acesse [fonts.google.com](https://fonts.google.com)
2. Escolha uma fonte que combine com seu personagem
3. Clique em "Get font" → "Get embed code"
4. Copie a tag `<link>` e cole no `<head>` do HTML
5. Use o nome da fonte no `font-family` do CSS

## Dúvidas frequentes

**Posso usar `<style>` dentro do HTML?**
Não. Todo o CSS deve estar no arquivo `estilo.css`.

**Posso mudar os IDs e classes do HTML?**
Não. Eles precisam existir como estão para o CSS funcionar.

**Posso adicionar novas tags no HTML?**
Sim, com moderação. Mas garanta que os elementos originais continuem intactos.

**Posso usar CSS que ainda não vimos em aula?**
Sim! Pesquisar é parte do aprendizado. Mas garanta que os requisitos obrigatórios estão presentes.
